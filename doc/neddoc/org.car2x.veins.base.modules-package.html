BaseLayer simple module

Base module for all layer modules. Defines the in and out gates
BaseMacLayer simple module

Base module for every mac layer module
BaseMobility simple module

Module which is responsible for mobility related information like position and movement BaseMobility itself defines a static mobility pattern (means only a position, no movement). Mobility modules which extend from this module will define more complex movement patterns.
BaseWorldUtility simple module

Basic utility module for the whole network. Provides utility methods and information used by the whole network as well as simulation wide black board functionality.
