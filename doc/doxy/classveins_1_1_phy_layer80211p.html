overrideprotectedvirtual

Attach a signal to the given AirFrame.

The attached Signal corresponds to the IEEE 802.11p standard. Parameters for the signal are passed in the control info. The indicated power levels are set up on the specified center frequency, as well as the neighboring 5MHz.

Note
The control info must be of type MacToPhyControlInfo11p
