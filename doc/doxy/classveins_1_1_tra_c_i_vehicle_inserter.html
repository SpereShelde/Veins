Detailed Description

Uses the TraCIScenarioManager to programmatically insert new vehicles at the TraCI server.

This is done whenever the total number of active vehicles drops below a given number.

See the Veins website for a tutorial, documentation, and publications .

Author
Christoph Sommer, David Eckhoff, Falko Dressler, Zheng Yao, Tobias Mayer, Alvaro Torres Cortes, Luca Bedogni
