71 /**
72  * @brief The basic Decider class
73  *
74  * The Deciders tasks are:
75  * 1. decide which packets should be handed up to the MAC Layer (primary task)
76  * 2. decide whether the channel is busy/idle at a time point or
77  * during a time interval (channel sensing)
78  *
79  * BasePhyLayer hands every receiving AirFrame several times to the
80  * "processSignal()"-function and is returned a time point when to do so again.
81  *
82  * @ingroup decider
83  */
