38 /**
39  * @brief Base module for all mobility modules.
40  *
41  * It does not provide mobility at all, so you can use
42  * it if you only want to simulate static networks.
43  *
44  * BaseMobility provides random placement of hosts and display
45  * updates.
46  * Change notifications about position changes are also posted to the
47  * Blackboard.
48  *
49  * Another service provided by BaseMobility is border handling. If a
50  * host wants to move outside the playground, this situation has to be
51  * handled somehow. BaseMobility provides handling for the 4 most
52  * common ways for that: reflection, wrapping, random placement and
53  * raising an error. The only thing you have to do is to specify your
54  * desired border handling in fixIfHostGetsOutside and call it in
55  * makeMove.
56  *
57  * For most mobility modules the only two functions you need to
58  * implement to define your own mobility module are makeMove and
59  * fixIfHostGetsOutside..
60  *
61  * This file contains 3D implementations from Michael Swigulski.
62  *
63  * @ingroup mobility
64  * @ingroup baseModules
65  * @author Daniel Willkomm, Andras Varga
66  */

291  /** @brief Main border handling function
292  *
293  * This function takes the BorderPolicy and all variables to be
294  * modified in case a border is reached and invokes the appropriate
295  * action. Pass dummy variables if you do not need them.
296  *
297  * The supported border policies are REFLECT, WRAP, PLACERANDOMLY, and
298  * RAISEERROR.
299  *
300  * The policy and stepTarget are mandatory parameters to
301  * pass. stepTarget is used to check whether the host actually moved
302  * outside the playground.
303  *
304  * Additional parameters to pass (in case of non atomic movements) can
305  * be targetPos (the target the host is moving to) and step (the size
306  * of a step).
307  *
308  * Angle is the direction in which the host is moving.
309  *
310  * @param policy BorderPolicy to use
311  * @param stepTarget target position of the next step of the host
312  * @param targetPos target position of the host (for non atomic movement)
313  * @param step step size of the host (for non atomic movement)
314  * @param angle direction in which the host is moving
315  *
316  * @return true if host was outside, false otherwise.
317  */
318  bool handleIfOutside(BorderPolicy, Coord&, Coord&, Coord&, double&);
