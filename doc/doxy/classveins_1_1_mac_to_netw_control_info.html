Detailed Description

Stores control information from mac to upper layer.

Holds the bit error rate of the transmission as well as the MAC address of the last hop.

Author
Karl Wessel

static cObject* const veins::MacToNetwControlInfo::setControlInfo ( cMessage *const  pMsg,
const LAddress::L2Type &  pSrcAddr 
)
inlinestatic

Attaches a "control info" structure (object) to the message pMsg.

This is most useful when passing packets between protocol layers of a protocol stack, the control info will contain the source MAC address.

The "control info" object will be deleted when the message is deleted. Only one "control info" structure can be attached (the second setL3ToL2ControlInfo() call throws an error).

Parameters
pMsgThe message where the "control info" shall be attached.
pSrcAddrThe MAC address of the message sender.
