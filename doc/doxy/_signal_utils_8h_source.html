42 /**
43  * @brief check if the summed power of interfererFrames's signals at freqIndex is below a given threshold.
44  *
45  * Only considers the signals active at time now and ignores the AirFrame given as exclude.
46  *
47  * This function will apply analogue models attached to the interfererFrames's signals.
48  * It will skip applying analogue models once it is clear that the interferer signals are below threshold.
49  * This is known as "thresholding" or some sort of short-circuit evaluation.
50  *
51  * TODO: apply analogue models one by one instead of group-wise.
52  * Currently assumes that all signals have the same analogue model (classes) attached to them.
53  * Each model is applied for all signals before the next check for the threshold is performed.
54  * This can be optimized to minimize the number of analog models needed to be applied.
55  */
56 bool VEINS_API isChannelPowerBelowThreshold(simtime_t now, AirFrameVector& interfererFrames, size_t freqIndex, double threshold, AirFrame* exclude = nullptr);
